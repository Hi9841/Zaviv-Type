# HyperType Logo Assets

Approved logo mark only — no wordmark or tagline.

## Contents
- 01_Master: 2048px transparent master PNG files
- 02_Color_Variants: transparent color variants
- 03_PNG_Sizes: common export sizes from 32px to 1024px
- 04_App_Icons: square app icon treatments for dark, purple, light, and black backgrounds
- 05_Vector_SVG: scalable SVG color variants

## Recommended use
- Main branding: `hypertype-logo-gradient-clean-2048.png`
- Dark UI / hero use: `hypertype-logo-gradient-glow-2048.png`
- Small sizes: use the flat `brand-purple`, `charcoal`, or `white` variants
- App stores: choose one of the square files in `04_App_Icons`

All exported PNG logo files use real transparency.
